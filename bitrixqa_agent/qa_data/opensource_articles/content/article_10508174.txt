Категория: Битрикс24

ТЕМА: Расчет прибыли в Битрикс24 с помощью роботов и пользовательских полей

ПРОБЛЕМА:
Для эффективного управления продажами и анализа рентабельности необходимо иметь точные данные о прибыли сделок. Вручную рассчитывать прибыль по каждой сделке трудоемко и чревато ошибками.

РЕШЕНИЕ:
1.  Настройка пользовательских полей:
1.    *   Создать два числовых поля: "Затраты" и "Прибыль".
2.    *   Добавить эти поля в карточку сделки.
3.    *   Сделать поле "Затраты" обязательным для заполнения со второй стадии воронки продаж, либо установить значение по умолчанию 0.
2.  Настройка робота "Изменение документа":
1.    *   Добавить робота на вторую стадию воронки продаж.
2.    *   В поле "Прибыль" установить формулу: `= {{Сумма}} - {{Затраты}}`.

ВАЖНО:
1.   Корректный расчет прибыли зависит от своевременного заполнения поля "Затраты".
2.   Если поле "Затраты" не будет заполнено при переходе сделки на вторую стадию, прибыль не рассчитается.
3.   После настройки робот будет автоматически рассчитывать прибыль при выполнении условий.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Пользовательские поля: Тип "Число".
2.   Робот: "Изменение документа".
3.   Формула расчета прибыли: `= {{Сумма}} - {{Затраты}}`.
