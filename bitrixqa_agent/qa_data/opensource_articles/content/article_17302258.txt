ТЕМА: Изменение языка интерфейса в Битрикс24

ПРОБЛЕМА:
Пользователю может требоваться изменить язык интерфейса ввиду личных предпочтений, работы с международными командами или других причин.

РЕШЕНИЕ:
1. Пользователь может самостоятельно изменить язык интерфейса в облачной версии Битрикс24.
2. Для этого необходимо на любой странице портала в его нижней части найти кнопку с текущим языком.
3. Нажатие на эту кнопку откроет список доступных языков, из которого можно выбрать нужный.

ВАЖНО:
- Изменение языка интерфейса доступно только в облачной версии Битрикс24.
- Язык интерфейса в десктопных приложениях (Windows, macOS, Linux) синхронизируется с языком операционной системы.
- По умолчанию язык портала определяется доменной зоной (например, .ru — русский язык).

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
- Функция смены языка доступна для каждого сотрудника индивидуально.
