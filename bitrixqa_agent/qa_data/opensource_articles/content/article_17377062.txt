ТЕМА: Интеграция видеоконтента из YouTube (Shorts и обычные видео) на сайт.

ПРОБЛЕМА: Как эффективно использовать видеоконтент из разных источников, включая популярные YouTube Shorts, для наполнения своего сайта и привлечения аудитории.

РЕШЕНИЕ:
1.  Для обычных видео с YouTube:
1.    *   Скопировать URL видео.
2.    *   В редакторе сайта добавить блок "Видео".
3.    *   В настройках блока "Видео" вставить скопированный URL.
4.    *   Настройть параметры запуска видео (при необходимости).
5.    *   Сохранить настройки.
2.  Для YouTube Shorts:
1.    *   Добавить Shorts на сайт так же, как и обычное видео (скопировать URL, добавить блок "Видео", вставить URL).
2.    *   Для настройки пропорций: перейти в раздел "Дизайн", нажать на видео.
3.    *   В открывшемся слайдере настроек выбрать опцию "Вертикальное, 9:16" для отображения в формате YouTube Shorts.
4.    *   Настроить масштаб видео.
5.    *   Переопубликовать сайт для применения изменений.

ВАЖНО:
1.   Возможность настройки пропорций для YouTube Shorts позволяет отображать их в привычном вертикальном формате.
2.   После внесения изменений в настройки видео (особенно пропорций Shorts) требуется переопубликация сайта.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Используется функционал добавления блока "Видео" на сайт.
2.   Предоставляются настройки соотношения сторон (9:16) и масштаба для Shorts.
