Категория: Битрикс24

ТЕМА: Перенос данных между порталами Битрикс24

ПРОБЛЕМА:
Пользователи, которые решили перейти на новый портал Битрикс24, хотят перенести существующую клиентскую базу, настройки и другую важную информацию со старого портала на новый.

РЕШЕНИЕ:
Перенос данных осуществляется путем экспорта информации с исходного портала и последующего импорта на целевой портал. Это касается следующих типов данных:
1.   Элементы CRM: лиды, сделки, контакты, компании. Формат: CSV.
2.   Товары: каталог товаров. Формат: экспорт в XLS, импорт в CSV (требуется конвертация).
3.   Отчеты CRM: из конструктора отчетов. Формат: CSV.
4.   Настройки CRM: пользовательские поля, стадии, бизнес-процессы и др. Формат: папка с файлами JSON.
5.   Задачи: список задач (без комментариев). Формат: экспорт в XLS, импорт в CSV (требуется конвертация).
6.   Отчеты по задачам: Формат: CSV.
7.   Базы знаний, сайты и магазины: Формат: папка с файлом JSON.
8.   Бизнес-процессы: шаблоны. Формат: BPT.

ВАЖНО:
1.   При переносе настроек CRM может произойти удаление лидов и сделок на новом портале. Рекомендуется выгрузить их предварительно.
2.   Комментарии к задачам не переносятся.
3.   Форматы файлов для экспорта и импорта могут различаться (например, XLS и CSV), потребуется конвертация.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Конвертация XLS в CSV:
2.    *   Через функцию "Сохранить как" в Excel, выбрав формат CSV.
3.    *   С помощью онлайн-конвертеров документов.
4.   Пути доступа к функциям экспорта/импорта:
5.    *   CRM: Настройки (⚙️) -> Экспорт/Импорт.
6.    *   Товары: CRM -> Товары и склады -> Каталог товаров -> Настройки (⚙️) -> Excel/Импорт товаров.
7.    *   Отчеты CRM: CRM -> Аналитика -> Конструктор отчетов -> Экспортировать/Импортировать.
8.    *   Настройки CRM: CRM -> Настройки (⚙️) -> Отраслевые сценарии -> Экспортировать/Импортировать.
9.    *   Задачи:  Задачи и проекты -> Настройки (⚙️) -> Экспорт/Импорт списка.
10.    *   Отчеты по задачам: Задачи и проекты -> Отчеты -> Экспортировать/Импортировать.
11.    *   Сайты/Магазины/База знаний: Сайты и Магазины/База знаний -> Экспорт/Настройки (⚙️) -> Импорт.
12.    *   Бизнес-процессы: Автоматизация -> Бизнес-процессы -> Редактирование шаблона -> Экспорт/Импорт.
