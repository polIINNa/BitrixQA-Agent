Категория: Битрикс24

ТЕМА: Системные поля в CRM Битрикс24

ПРОБЛЕМА:
В CRM-системе Битрикс24 существует множество полей для хранения информации о клиентах, сделках, товарах и других элементах. Пользователям необходимо понимать, какие поля являются системными (предустановленными и неизменяемыми), какие специфическими для определенных элементов, и как их использовать для эффективной работы.

РЕШЕНИЕ:
Использовать системные поля, которые предоставляются Битрикс24 "из коробки", для хранения ключевой информации. Эти поля нельзя удалять, но некоторые из них (отмеченные звездочкой) можно настраивать в "Справочнике CRM" или использовать для автоматизации. При необходимости можно создавать пользовательские поля для уникальных нужд компании.

ВАЖНО:
1.   Системные поля являются основой для большинства операций в CRM.
2.   Некоторые системные поля (например, "Стадия", "Обращение", "Источник", "Тип компании/контакта") можно настроить для более точного соответствия бизнес-процессам.
3.   Поля "Название элемента", "Сумма и валюта", "Клиент", "Телефон", "E-mail", "ID", "Ответственный", "Когда создан", "Кем создан" и другие являются базовыми для большинства элементов CRM.
4.   Специализированные поля для Лидов, Сделок, Товаров, Компаний, Контактов, Счетов и Предложений позволяют детализировать информацию по каждому типу сущностей.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Общие поля: Название элемента, Стадия, Сумма и валюта, Клиент, Обращение, Телефон, E-mail, Сайт, Мессенджер, Источник, Дополнительно об источнике, ID, Ответственный, Наблюдатели, Тип компании/контакта, Комментарий, UTM-метки, Сквозная аналитика, Товары, Доступен для всех, Реквизиты вашей компании, Когда создан, Когда обновлен, Кем создан, Кем обновлен, Последняя активность, Дата последней коммуникации, Дата завершения.
2.   Поля для Лида: Фамилия, Имя, Отчество, Клиент (повторный), Дата рождения, Должность, Адрес, Дата создания, Дата изменения, Дополнительно о стадии.
3.   Поля для Сделки: Тип сделки, Дата создания, Дата начала, Вероятность, Повторяемость.
4.   Поля для Товаров: Наименование, Детальное описание, Группа товаров, Маркировка товаров, Видимость товара, Разделы, Идентификатор, Дата изменения, Кто изменил, Дата создания, Кто создал, Идентификатор инфоблока, Активность, Индекс сортировки, Описание для анонса, Внешний код.
5.   Поля для Компаний: Название компании, Контактный номер, E-mail, Сфера деятельности, Годовой оборот, Контакт, Реквизиты, Количество сотрудников, Банковские реквизиты.
6.   Поля для Контактов: Фамилия, Имя, Отчество, Контактный телефон, E-mail, Участвует в экспорте контактов.
7.   Поля для Счетов: Дата выставления, Срок оплаты, Сделка, Когда передвинут, Кем передвинут, Предыдущая стадия, Номер счета, Предложение.
8.   Поля для Предложений: № предложения, Тема, Файлы, Лид, Сделка, Дата начала, Актуально до, Содержание, Условия, Предложение закрыто.
