Категория: Битрикс24

ТЕМА: Усиление безопасности аккаунта Битрикс24 с помощью двухфакторной аутентификации

ПРОБЛЕМА:
Недостаточная защита аккаунтов Битрикс24 от несанкционированного доступа, когда злоумышленнику достаточно знать только логин и пароль.

РЕШЕНИЕ:
1.  Включение двухфакторной аутентификации (2FA): Пользователь вводит логин и пароль, а затем одноразовый код из специального приложения.
2.  Обязательная 2FA для всех сотрудников (Администратор):
1.    *   Администратор настраивает 2FA для своего аккаунта.
2.    *   Включает опцию "Обязательно для всех сотрудников" в настройках безопасности, устанавливая временной лимит на включение 2FA.
3.    *   Опционально настраивает отправку кодов авторизации в чат Битрикс24 для уведомлений.
3.  Самостоятельное включение 2FA (Сотрудник):
1.    *   Пользователь переходит в личный профиль, раздел "Безопасность", и нажимает "Подключить".
2.    *   Скачивает мобильное приложение, например, Bitrix24 OTP (можно использовать любое TOTP-совместимое).
3.    *   Настраивает приложение, сканируя QR-код или вводя данные вручную.
4.    *   При каждом входе использует логин, пароль и одноразовый код.

ВАЖНО:
1.   Для корректной работы TOTP-кодов время на мобильном устройстве и компьютере должно совпадать.
2.   Необходимо сохранить резервные коды на случай потери или недоступности мобильного устройства.
3.   В случае смены мобильного устройства, на котором было установлено приложение для генерации кодов, 2FA нужно настроить заново.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Используется алгоритм TOTP (Time-based One-Time Password Algorithm).
2.   Требуется мобильное приложение для генерации одноразовых кодов (например, Bitrix24 OTP).
3.   Резервные коды одноразовые.

1.   Как администратору сделать двухфакторную аутентификацию обязательной для всех сотрудников?

3.   Можно ли использовать другие приложения для двухфакторной аутентификации, кроме Bitrix24 OTP?
4.   Как настроить двухфакторную аутентификацию на новом телефоне?
5.   Как сотрудники получают уведомления о попытке входа в их аккаунт?
