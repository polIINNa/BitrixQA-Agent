Категория: Битрикс24

ТЕМА: Установка десктопного приложения Битрикс24 на macOS и Windows

ПРОБЛЕМА: Пользователям необходимо установить десктопное приложение Битрикс24 для получения дополнительных функций и удобства работы. Процесс установки может отличаться в зависимости от операционной системы.

РЕШЕНИЕ:
1.  Скачивание: Загрузите установочный файл приложения Битрикс24 с официального сайта Битрикс24. Приложение доступно для macOS, Windows и Linux.
2.  Установка на macOS:
1.    *   Дважды кликните на скачанный DMG-файл.
2.    *   Перенесите иконку приложения в папку "Applications".
3.    *   Подтвердите запуск приложения и примите лицензионное соглашение.
3.  Установка на Windows:
1.    *   Дважды кликните на скачанный EXE-файл.
2.    *   Выберите папку для установки, при необходимости, используя кнопку "Обзор".
3.    *   Дождитесь завершения установки и нажмите "Готово".
4.  Авторизация: После установки откройте приложение и войдите в свой аккаунт Битрикс24.

ВАЖНО:
1.   Приложение Битрикс24 предлагает дополнительные функции, такие как запись звонков, выбор фона и двусторонняя синхронизация диска.
2.   Для работы "эластичного диска" с MSI-дистрибутивом может потребоваться установка драйвера Dokan (для Windows) или macFUSE (для macOS).
3.   Приложение может работать некорректно с файловой системой exFAT.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Системные требования:
2.    *   macOS: версия 11 и выше.
3.    *   Windows: версия 10 и выше, Windows Server 2016 и выше.
4.    *   64-битные версии Windows для приложения Битрикс24 от версии 15.
5.   [Ссылка на скачивание драйвера Dokan/macFUSE]
