Категория: Битрикс24

ТЕМА: Кастомизация главного меню в Битрикс24

ПРОБЛЕМА:
Пользователи и администраторы Битрикс24 нуждаются в гибкой настройке главного меню для оптимизации рабочего процесса и быстрого доступа к необходимым инструментам. Необходимо иметь возможность персонализировать меню под индивидуальные нужды каждого сотрудника и унифицировать его для всей компании, а также скрывать неиспользуемые функции.

РЕШЕНИЕ:
Для пользователей:
1.   Изменение сортировки пунктов меню путем перетаскивания.
2.   Добавление новых ссылок на инструменты Битрикс24 или внешние сайты.
3.   Смена основного инструмента, что влияет на порядок пунктов в меню.
4.   Возврат к меню по умолчанию.

Для администраторов:
1.   Установка единого меню для всех сотрудников (или только для новых).
2.   Сокрытие неиспользуемых инструментов из главного меню для всей компании.

ВАЖНО:
1.   Главное меню отображается в свернутом виде по умолчанию. Для развертывания нажмите на иконку "Меню" (≡).
2.   Настройка меню осуществляется через "Настройки" > "Настройка левого меню".
3.   Единое меню для всей компании доступно не на всех тарифных планах.
4.   Скрытые инструменты будут полностью отключены в Битрикс24.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Добавить ссылку: "Настройки" > "Настройка левого меню" > "Добавить ссылку в меню".
2.   Сменить главный инструмент: "Настройки" > "Сменить главный инструмент" > "Подтвердить".
3.   Вернуть меню по умолчанию: Кнопка "Вернуть меню по умолчанию".
4.   Установить меню для всех: "Настройки" > "Установить мое меню для всех".

1.   Как изменить сортировку пунктов в меню?
2.   Каким образом добавить ссылку на внешний сайт в главное меню?
3.   Можно ли восстановить стандартное меню, если оно было сильно изменено?
4.   Какие инструменты могут быть скрыты администратором?
5.   На каких тарифах Битрикс24 доступна функция установки единого меню для всей компании?
