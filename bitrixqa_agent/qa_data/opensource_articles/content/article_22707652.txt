Категория: Битрикс24

ТЕМА: Работа с файлами в пространстве коллаборации "Коллаба" в Битрикс24

ПРОБЛЕМА:
Пользователи (сотрудники и гости) сталкиваются с трудностями при поиске, добавлении, редактировании и управлении файлами, хранящимися в пространствах коллаборации (коллабах). Не всегда ясно, где хранятся файлы и как получить к ним доступ.

РЕШЕНИЕ:
1.  Для сотрудников и создателей коллабы:
1.    *   Поиск файлов:
2.        *   Через раздел "Мессенджер" -> "Коллабы" -> Выбрать коллабу -> "Файлы".
3.        *   Через раздел "Диск" -> "Мой диск" -> Папка с названием коллабы.
4.    *   Хранение: Все файлы, добавленные в чаты, события и задачи коллабы, автоматически попадают в общую папку "Загруженные файлы" внутри коллабы и на "Диск".
5.    *   Создание/Загрузка: Нажать "Добавить" в разделе "Файлы" коллабы, выбрать тип файла (документ, таблица, презентация) или загрузить с компьютера. Возможно создание папок для организации.
6.    *   Управление: Использовать меню (≡) рядом с файлом для действий (поделиться, переместить, скачать, удалить).
7.    *   Настройка прав доступа: Только владелец коллабы может настроить права (просмотр, редактирование, удаление) через "Настройки (⚙️) > Настройка прав".

2.  Для гостей коллабы:
1.    *   Поиск файлов:
2.        *   Через раздел "Мессенджер" -> "Коллабы" -> Выбрать коллабу -> "Файлы".
3.        *   Через раздел "Файлы" -> Папка с названием коллабы.
4.    *   Создание/Загрузка: Нажать "Добавить" в разделе "Файлы" коллабы, выбрать тип файла или загрузить с компьютера. Возможно создание папок.
5.    *   Управление: Использовать меню (≡) рядом с файлом для действий (скачать, переместить и т.д.). Функционал может быть ограничен правами гостя.

ВАЖНО:
1.   "Коллаба" — это выделенное пространство для совместной работы, включая обмен файлами, с внешними пользователями.
2.   Все файлы коллабы находятся в одном централизованном месте, доступном через разные разделы Битрикс24.
3.   Сотрудники и гости имеют разные уровни доступа и возможности управления файлами.
4.   Владелец коллабы играет ключевую роль в управлении доступом к файлам.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Файлы коллабы синхронизируются с "Диском" пользователя.
2.   Функции создания файлов включают документы, таблицы и презентации (аналогично инструментам Диска).
3.   Присутствуют функции поиска по названию и фильтрации файлов.
