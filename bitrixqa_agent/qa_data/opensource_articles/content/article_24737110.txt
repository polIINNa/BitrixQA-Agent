ТЕМА: Использование CoPilot в коробочной версии Битрикс24: бесплатные запросы и их особенности

ПРОБЛЕМА: Компании, использующие коробочную версию Битрикс24, имеют ограниченное количество бесплатных запросов к AI-помощнику CoPilot. Непонимание механики начисления и использования этих запросов может привести к их преждевременному исчерпанию или неэффективному использованию.

РЕШЕНИЕ:
1.  Ознакомление с бесплатным пакетом: Понять, что в каждой коробочной версии Битрикс24 предоставляется ежемесячный бесплатный пакет запросов к CoPilot.
2.  Определение количества запросов: Уточнить количество бесплатных запросов, которое зависит от конкретной редакции коробочной версии Битрикс24.
3.  Понимание механики расчёта: Знать, что пакет запросов обновляется 1-го числа каждого месяца и неиспользованные запросы сгорают. При смене редакции происходит пересчёт.
4.  Учёт разных сценариев использования: Понимать, что каждый тип взаимодействия с CoPilot (обращение в ленте, чат, обработка звонка, речевая аналитика, скрипты продаж, потоки) может тарифицироваться как один или несколько запросов.
5.  Управление доступом: При необходимости можно отключить CoPilot в определенных разделах Битрикс24 через настройки администратора для экономии запросов.
6.  Приобретение дополнительных запросов: Если бесплатных запросов не хватает, можно приобрести "бусты CoPilot" на год.

ВАЖНО:
1.   Неиспользованные бесплатные запросы сгорают в конце месяца.
2.   При смене редакции Битрикс24 количество запросов пересчитывается.
3.   Запросы суммируются для всей компании.
4.   При покупке бустов CoPilot бесплатный пакет запросов становится недоступен (действуют только бусты).
5.   Бесплатные запросы не начисляются, если есть неиспользованные бусты.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Начисление: Ежемесячно, 1-го числа.
2.   Обнуление: Неиспользованные запросы сгорают.
3.   Пересчёт при смене редакции: [Бесплатный пакет в новой редакции] – [Количество потраченных запросов] = [Остаток].
4.   Отключение: Настройки → Настройки CoPilot (переключатели для разделов).
5.   Приобретение: Покупка "бустов CoPilot" на год.
