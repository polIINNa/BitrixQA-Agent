Категория: Битрикс24

ТЕМА: Настройка внешнего вида Битрикс24

ПРОБЛЕМА:
Сотрудникам и компаниям может требоваться изменить стандартный внешний вид Битрикс24, чтобы он соответствовал корпоративному стилю, повышал узнаваемость бренда или просто соответствовал индивидуальным предпочтениям пользователей.

РЕШЕНИЕ:
1.  Установка стандартной темы:
1.    *   Нажать на фотографию пользователя в правом верхнем углу.
2.    *   Выбрать опцию "Сменить фон".
3.    *   Просмотреть доступные темы и выбрать понравившуюся.
4.    *   Нажать "Сохранить".
2.  Установка корпоративной темы (администратором):
1.    *   После выбора стандартной темы, администратор может поставить галочку "Включить всем по умолчанию", чтобы применить ее ко всем пользователям.
3.  Добавление собственной темы (для администратора):
1.    *   Перейти в настройки темы по пути: Нажмите на вашу фотографию -> Сменить фон -> Своя тема.
2.    *   Загрузить изображение в формате PNG, JPG или GIF.
3.    *   При необходимости, указать цвет фона (для полупрозрачных изображений) и цвет текста, чтобы обеспечить читаемость.

ВАЖНО:
1.   Обычный сотрудник может изменить тему оформления только для своего личного профиля.
2.   Администратор Битрикс24 может установить единую тему оформления для всех сотрудников.
3.   Функция установки корпоративной темы может быть доступна не на всех тарифных планах.
4.   Темы можно менять неограниченное количество раз.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Поддерживаемые форматы изображений для собственных тем: PNG, JPG, GIF.
2.   Максимальный размер файла изображения: 20 Мб.
3.   Максимальное разрешение изображения: 5000x5000 пикселей.
4.   Чрезмерно большие изображения могут замедлить работу Битрикс24.
5.   Есть возможность удалить загруженное собственное изображение.

3.   Какие требования к размеру и разрешению загружаемых изображений для корпоративной темы?
4.   Где найти настройки смены фонового изображения в Битрикс24?
