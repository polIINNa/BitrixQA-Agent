Категория: Битрикс24

ТЕМА: Документация и обучение по Битрикс24 в коробке

ПРОБЛЕМА:
Пользователям и администраторам коробочной версии Битрикс24 может быть сложно ориентироваться в большом объеме информации, необходимой для эффективной работы и администрирования системы.

РЕШЕНИЕ:
Организовать доступ к информации через различные каналы:
1.  Учебные курсы: для освоения методов работы с программным продуктом.
2.  Портал "Поддержка24": содержит пользовательские статьи, применимые как для облачной, так и для коробочной версии, а также отдельный раздел с особенностями "Битрикс24 в коробке".
3.  Документация: для понимания основ изменения и администрирования корпоративного портала.

ВАЖНО:
1.   Пользовательские задачи: Статьи с портала "Поддержка24" актуальны для коробочной версии.
2.   Администраторские задачи: Для администраторов коробочной версии есть специализированные разделы, охватывающие установку, настройку, основы администрирования, работу администратора КП и администратора системы.
3.   Документация для разработчиков: Доступна в онлайн-версии (рекомендуется как наиболее актуальная) и в формате CHM (обновляется периодически).
4.   CHM-файлы: При проблемах с отображением содержимого CHM-файлов, необходимо снять блокировку в свойствах файла.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Онлайн-документация: Рекомендуется для получения самой свежей информации.
2.   Оффлайн-документация (CHM): Доступна для скачивания, но может отставать по актуальности.
3.   API: Описание API старого ядра Bitrix Framework и нового ядра D7.
4.   Интеграция: Информация по связке портала с 1С:ЗУП ред. 3.0.
