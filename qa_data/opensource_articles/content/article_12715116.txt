Категория: Битрикс24

ТЕМА: Обновление требований к серверному окружению для модуля "Веб-мессенджер" (im) в коробочной версии Битрикс24.

ПРОБЛЕМА:
С 2021 года для корректной работы модуля "Веб-мессенджер" (im) в коробочных версиях Битрикс24 требуется настройка сервера очередей Push and Pull. Прекращается поддержка старых версий локальных серверов очередей, что может привести к неработоспособности чатов и других инструментов, требующих мгновенного взаимодействия.

РЕШЕНИЕ:
1. Проверить текущую настройку сервера очередей в административном разделе: Настройки > Настройки продукта > Настройки модулей > Push and Pull.
2. Убедиться, что выбран вариант "облачный сервер «1С-Битрикс»" или "локальный сервер Bitrix Push server".
3. Если используется старая версия локального сервера очередей (Nginx-PushStreamModule 0.3.4/0.4.0 или Bitrix Push server 1.0), перейти на Bitrix Push server 2.0.
4. При необходимости установить облачный сервер очередей «1С-Битрикс» или локальный Bitrix Push server 2.0, следуя предоставленным инструкциям.

ВАЖНО:
- После обновления модуля "Веб-мессенджер" (im) 20.400.0 поддержка чатов без настроенного сервера очередей будет прекращена.
- Необходимо перейти на Bitrix Push server 2.0, если используется старая версия локального сервера очередей.
- Сервер очередей Push and Pull важен для мгновенного взаимодействия во многих инструментах Битрикс24, включая Задачи, Календари, Ленту Новостей, Чат и другие.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
- Модуль: Веб-мессенджер (im)
- Требование: Настроенный сервер очередей Push and Pull.
- Рекомендуемая локальная версия сервера очередей: Bitrix Push server 2.0.
- Старые поддерживаемые версии локальных серверов очередей: Nginx-PushStreamModule 0.3.4/0.4.0, Bitrix Push server 1.0 (поддержка прекращена осенью 2021 г.).
