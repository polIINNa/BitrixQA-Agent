Категория: Битрикс24

ТЕМА: Управление встречами с клиентами с помощью "Свободных слотов" в Битрикс24 CRM

ПРОБЛЕМА:
Сотрудники, работающие с клиентами, сталкиваются с трудностями в согласовании удобного времени для встреч. Процесс назначения встреч может быть долгим и трудоемким, требуя множества звонков и переписок для поиска свободного окна в расписании.

РЕШЕНИЕ:
Инструмент "Свободные слоты" в Битрикс24 CRM позволяет клиентам самостоятельно выбирать удобное время для встречи из предложенного расписания сотрудника.
1.  Настройка расписания: В карточке сделки сотрудник настраивает дни и время, когда он доступен для встреч, разделяя их на равные промежутки.
2.  Приглашение коллег: При необходимости можно добавить коллег на встречу. Расписание будет учитывать их занятость.
3.  Отправка ссылки клиенту: Ссылка на настроенное расписание отправляется клиенту (например, по SMS через контакт-центр).
4.  Выбор слота клиентом: Клиент выбирает удобный день и время для встречи.
5.  Автоматическое создание события: Выбранная встреча автоматически появляется в календаре Битрикс24 и в виде "Дела" в карточке CRM.

ВАЖНО:
1.   Для отправки слотов через SMS необходимо подключить соответствующий канал в контакт-центре.
2.   Ссылка на расписание активна 7 дней.
3.   Назначенную встречу можно отредактировать (название, добавить участников), но нельзя изменить время.
4.   Отказаться от встречи можно через "Дело" в CRM, клиент получит уведомление.
5.   Все действия клиента (просмотр расписания, назначение встречи) отражаются в карточке CRM.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Канал связи: SMS (через контакт-центр).
2.   Интеграция: Календарь Битрикс24, карточка CRM (создание "Дела").
3.   Видеоконференции: Интегрируется с Битрикс24.Видеоконференции.
