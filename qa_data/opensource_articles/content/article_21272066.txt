Категория: Битрикс24

ТЕМА: Создание и настройка Потоков в Битрикс24

ПРОБЛЕМА:
Командам необходимо эффективно собирать и распределять задачи внутри отдела или проекта, обеспечивая равномерную нагрузку или гибкий подход к выполнению. Требуется инструмент, который поможет систематизировать работу над задачами.

РЕШЕНИЕ:
1.  Перейдите в раздел "Задачи и Проекты > Потоки".
2.  Выберите способ создания Потока:
1.    *   Создайте новый Поток с нуля.
2.    *   Выберите готовый пример из предложенных шаблонов.
3.  Заполните информацию о Потоке:
1.    *   Укажите название и описание.
2.    *   Определите, кто может добавлять задачи в Поток.
3.    *   Установите срок выполнения задач по умолчанию.
4.  Настройте распределение задач:
1.    *   По очереди: Автоматическое назначение задач следующим сотрудникам в списке для равномерной нагрузки.
2.    *   Вручную: Задачи назначаются выбранному сотруднику (например, координатору) для последующего распределения.
3.    *   Самостоятельно: Сотрудники сами берут задачи, когда готовы приступить.
4.    *   По правилам автоматизации: Задачи создаются и назначаются автоматически с помощью роботов или бизнес-процессов, с определением исполнителя по заданным параметрам (например, "Ответственный за сделку" или "Создатель лида").
5.  Настройте дополнительные опции: Разрешение изменения сроков, уведомления, отправка на проверку.
6.  Выберите проект для Потока или позвольте системе создать его автоматически.
7.  Настройте контроль: Назначьте администратора Потока, который будет получать уведомления и следить за эффективностью команды (при снижении эффективности Поток станет красным).

ВАЖНО:
1.   Любой сотрудник может создать новый Поток по умолчанию. Администратор Битрикс24 может ограничить это право в настройках задач.
2.   Вы можете временно приостановить работу Потока, выбрав опцию "Выключить" в меню.
3.   Привязать несколько Потоков можно к одному проекту.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Распределение задач: По умолчанию, вручную, самостоятельно, по правилам автоматизации.
2.   Система уведомлений: Для администратора Потока об эффективности и изменениях в задачах.
3.   Права доступа: Настраиваются администратором Битрикс24.
