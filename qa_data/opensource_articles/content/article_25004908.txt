Категория: Битрикс24

ТЕМА:
Создание и настройка открытой линии в Битрикс24

ПРОБЛЕМА:
Компании необходимо организовать эффективное общение с клиентами через различные каналы (мессенджеры, соцсети, онлайн-чат), автоматизировать рутинные задачи и контролировать качество работы менеджеров.

РЕШЕНИЕ:
1.  Создание Открытой линии:
1.    *   Перейти в раздел CRM > Клиенты > Контакт-центр.
2.    *   Выбрать нужный канал коммуникации (Telegram, ВКонтакте и т.д.).
3.    *   Нажать "Создать открытую линию".
4.    *   Важно: Каждый канал можно подключить только к одной открытой линии. Количество линий зависит от тарифа.

2.  Настройка Открытой линии:
1.    *   Очередь:
2.        *   Назначить ответственных сотрудников или отдел.
3.        *   Выбрать способ распределения обращений: равномерно, строго по очереди, одновременно всем.
4.        *   Настроить время до перехода обращения к следующему сотруднику (для равномерного и строго по очереди распределения).
5.        *   Включить проверку доступности оператора (учитывает увольнение, отпуск, онлайн-статус).
6.        *   Ограничить количество одновременных обращений на оператора.
7.        *   Определить, какие данные сотрудника увидит клиент (из профиля, замененные, скрытые).
8.        *   Включить проверку клиента по базе CRM (автоматическое создание лида/сделки или ручное, определение источника, назначение ответственного).
9.    *   Рабочее время:
10.        *   Настроить часовой пояс, режим работы, выходные и праздничные дни.
11.        *   Установить автоответ для сообщений, полученных в нерабочее время.
12.        *   Важно: Для виджета на сайте можно настроить расписание отдельно.
13.    *   Соглашения:
14.        *   Включить отправку предупреждения о сборе персональных данных для соблюдения законодательства.
15.    *   Автоматические действия:
16.        *   Настроить автоматические ответы на сообщения клиента (при первом обращении, на каждое новое).
17.        *   Использовать CRM-форму для запроса контактной информации.
18.        *   Установить время до отметки обращения как неотвеченного и действия в случае неответа.
19.        *   Настроить автоответ при завершении обращения.
20.        *   Определить время для оценки работы оператора после завершения диалога.
21.        *   Установить время до автозакрытия диалога при отсутствии ответа от клиента.
22.        *   Настроить действия после автозакрытия диалога.
23.        *   Подключить список быстрых ответов.
24.        *   Включить наблюдение за набором текста клиентом.
25.    *   Оценка качества:
26.        *   Настроить отправку запроса клиенту на оценку работы сотрудника.
27.        *   Разрешить оценку до окончания диалога, закрывать сессию после оценки, ограничить время на получение оценки.
28.    *   Чат-боты:
29.        *   Назначить ответственным чат-бота для первой обработки обращений.
30.        *   Выбрать конкретного чат-бота и настроить время его работы и перевода диалога на оператора.
31.        *   Важно: Новых чат-ботов можно установить из Битрикс24 Маркетплейс.
32.    *   KPI:
33.        *   Настроить KPI по времени ожидания первого и последующих ответов.
34.        *   Настроить уведомления о необработанных чатах.
35.        *   Важно: KPI можно отслеживать в разделе "Список диалогов".
36.    *   Прочее:
37.        *   Указать название открытой линии.
38.        *   Активировать линию, если она должна быть рабочей.
39.        *   Важно: Чтобы полностью остановить прием сообщений, необходимо отключить канал коммуникации.
40.        *   Включить сохранение истории сообщений.
41.        *   Настроить подтверждение при закрытии диалога.
42.        *   Выбрать язык уведомлений.
43.        *   Удалить открытую линию при необходимости (с удалением связанных чатов).

3.  Настройка прав доступа:
1.    *   В разделе "Настройки открытых линий > Права доступа" выбрать сотрудников, имеющих право изменять настройки и просматривать историю чатов.
2.    *   Важно: Настройки прав действуют для всех открытых линий.

ВАЖНО:
1.   Количество открытых линий зависит от вашего тарифа Битрикс24.
2.   Каждый канал коммуникации может быть подключен только к одной открытой линии.
3.   Для полной остановки приема сообщений необходимо отключить сам канал коммуникации, а не только открытую линию.
4.   Настройки прав доступа к открытым линиям применяются ко всем линиям.
5.   Отключенная открытая линия не прекращает прием сообщений с канала коммуникации – их просто не будет видно в Битрикс24.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   CRM > Клиенты > Контакт-центр – основной раздел для создания и управления открытыми линиями.
2.   Инструмент "Списки" используется для создания быстрых ответов.
3.   Настройки CRM-формы и чат-трекера интегрируются с процессом обработки обращений.
