ТЕМА: Интеграция видеоконтента из VK и Rutube на сайт Битрикс24

ПРОБЛЕМА:
Необходимость добавить видеоролики из популярных платформ VK и Rutube на сайт, созданный в Битрикс24, без необходимости предварительно скачивать и загружать файлы. Сделать сайт более динамичным и информативным для пользователей.

РЕШЕНИЕ:
1.  Получение ссылки на видео:
1.    *   VK: Найти видео, нажать "Еще" -> "Экспортировать" и скопировать прямую ссылку.
2.    *   Rutube: Скопировать ссылку на видео из адресной строки браузера.
2.  Добавление и настройка блока с видео:
1.    *   В разделе "Сайты и Магазины" -> "Сайты" выбрать нужный сайт и страницу.
2.    *   Нажать "Добавить блок" -> выбрать шаблон из вкладки "Видео".
3.    *   В режиме редактирования блока:
4.        *   Указать название ролика.
5.        *   Вставить ссылку на видео.
6.        *   Для VK: авторизоваться через VK ID (разрешить всплывающие окна в браузере).
7.        *   Для Rutube: вставить ссылку (авторизация не требуется).
3.  Настройка воспроизведения:
1.    *   Установить открытие видео во всплывающем окне.
2.    *   Для VK: включить автовоспроизведение (опционально).
3.    *   Для Rutube: указать секунду начала воспроизведения (опционально).
4.  Публикация: Сохранить изменения. Правки публикуются автоматически.

ВАЖНО:
1.   Добавлять видео могут только сотрудники с правами на редактирование контента сайта.
2.   Для интеграции с VK требуется авторизация через VK ID.
3.   Настройка воспроизведения (автовоспроизведение, время начала) зависит от платформы.
4.   Некоторые настройки, например, фоновое видео, могут не поддерживаться для Rutube.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
1.   Использование прямых ссылок на видеоконтент.
2.   Настройка блока "Видео" в конструкторе сайтов Битрикс24.
3.   Авторизация через VK ID требует разрешения всплывающих окон в браузере.
