ТЕМА: Пульс компании – Анализ использования Битрикс24

ПРОБЛЕМА:
Компании испытывают трудности с контролем и оценкой эффективности использования корпоративного портала Битрикс24 сотрудниками. Необходим инструмент для мониторинга активности, общего уровня вовлеченности и выявления наиболее активных сотрудников.

РЕШЕНИЕ:
Использовать модуль "Пульс компании" в Битрикс24, который предоставляет два инструмента:
1. Виджет "Пульс компании": Отображает текущее состояние использования портала, позволяя получить быстрый снимок активности.
2. Отчет "Пульс компании": Предоставляет детальную статистику по использованию различных инструментов Битрикс24 за выбранный период времени.

ВАЖНО:
- "Пульс компании" учитывает активность только внутренних сотрудников, исключая пользователей Экстранета.
- Данные в "Пульсе" обновляются с разным временным интервалом: моментально для конкретного пользователя, в течение минуты для графика "Сегодня" и в течение часа для графиков "Неделя" и "Месяц".
- Для оценки используются понятия:
    - Рейтинг: Список наиболее активных сотрудников, которые пользовались инструментами портала минимум один раз за выбранный период.
    - Активность: Суммарное количество действий, совершенных сотрудниками при использовании инструментов портала.
    - Вовлеченность: Процент сотрудников, использующих минимум четыре инструмента портала.
- Возможность переключения между отображением данных по отдельным пользователям и по компании в целом.
- При просмотре данных по конкретному сотруднику отображаются графики активности самого сотрудника, просматривающего сотрудника, отдела и компании.

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
- Переключение периодов времени: "Сегодня", "Неделя", "Месяц" для анализа динамики.
- Переключатель "Люди-Компания": Для детализации по отдельным сотрудникам или отделам.
- Инструменты для анализа: Социальная сеть, Лайки, Задачи и проекты, Чат и видео, Диск, Мобильная версия, CRM.
- Отображаемые метрики: Рейтинг, Активность, Вовлеченность.
